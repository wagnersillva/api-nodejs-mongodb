# Install

### Clone the repo

    git clone https://github.com/wagnersillva/api-nodejs-mongodb.git

### Install dependencies
    npm install

### Access folder 
    cd api-nodejs-mongodb

### To run server
    npm start

### Routes for test
    [GET] /products
    [POST] /products/create
    [POST] /image/upload
